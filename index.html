全方位的程式語言學習解決方案

Coala 程式語言學習中心提供全方位的免費程式學習資源，如線上課程與獨家練習系統，致力於培養、訓練臺灣未來程式設計人才，以解決目前臺灣軟體產業人才嚴重短缺的問題。

規劃完善的線上學習課程與練習系統

Codellege 是一個課程內容豐富、以實際練習為導向，擁有大量實作練習題的線上程式學習系統。提供一系列完整且經過專家所規劃的課程，如 Java 以及 C 語言，內容循序漸進、由淺入深。根據多年教學經驗的實證，大量的練習與實作就是學習程式語言最快的捷徑。使用者只要一步一步跟著 Codellege 內精心規劃的課程，不斷動手撰寫程式，就能最有效率地熟悉並且活用各種程式語言。

突破師生距離限制的遠距教學平台

TutorCloud 是一個具有高互動性的遠距程式教學平台，除了提供線上多人即時同步的程式撰寫與討論空間之外，更提供雲端程式編譯功能，使用者無需在自己的電腦上安裝任何額外的軟體，就可以直接透過瀏覽器進行程式編譯，並即時顯示執行結果。再透過即時互動白板、文件上傳與聊天室的輔佐，對於程式的討論與教學更加直覺便利。

企業與人才，老師與學生的媒合平台

MatchBoard 是一個提供程式相關媒合的平台，無論是企業徵才，或是程式學習需求，都可以利用 MatchBoard 來進行配對。企業甚至能夠搭配 Codellege 學習歷程，篩選、配對並找出適合的人才。有程式語言學習需求的使用者，也可以透過 MatchBoard 來尋找適合自己的指導者，同時再搭配 TutorCloud 的使用，以提升程式學習成效 。
